# CentOS
下载：MegaCli-8.07.10-1.noarch.rpm \
安装：rpm -ivh MegaCli-8.07.10-1.noarch.rpm \
使用：/opt/MegaRAID/MegaCli/MegaCli64 -PDList -aALL -NoLog \

安装依赖包：apt install libncurses5 \
使用：/opt/MegaRAID/MegaCli/MegaCli64 -PDList -aALL -NoLog \
